# Fuy App - Content & Media Pages Detailed Description

This document provides a comprehensive breakdown of every feature, button, and interaction available on the Media and Content-focused Screens.

---

## ðŸ“± Mobile Watch Screen
**File Reference:** `mobile/app/watch.tsx`

A TikTok/Reels-style full-screen video feed.

### 1. Feed Interaction
*   **Vertical Scroll:** Snaps to page height (`pagingEnabled`) to move between videos.
*   **Gestures:** Auto-plays current video. Tapping toggles play/pause (implied).

### 2. Overlay UI
*   **Right Action Bar:**
    *   **Avatar:** Circular profile pic. Tapping navigates to user profile.
    *   **Heart:** Like button with count. Toggles red/filled state.
    *   **Comment:** Opens comment sheet.
    *   **Share:** Opens share modal.
*   **Bottom Info:**
    *   **Username:** @handle.
    *   **Description:** Video title/caption.
    *   **Stats:** View count.
*   **Header:**
    *   **Back Button:** Returns to previous screen.
    *   **Mute Toggle:** Switch between Sound On/Off.

## ðŸ“± Mobile Stories Screen
**File Reference:** `mobile/app/stories.tsx`

A transient content viewer similar to Instagram Stories / Snapchat.

### 1. Grid View (Main)
*   **Header:** "Stories" title.
*   **Add Story:** Prominent card to create a new story ("Plus" icon).
*   **Story Grid:** 2-column grid of active stories from other users.
    *   **Preview:** Thumbnail of the first media item.
    *   **Meta:** User name and avatar overlay.

### 2. Story Viewer (Full Screen)
*   **Navigation:** Tap Left (Previous) / Right (Next) zones. Auto-advances after 5s.
*   **Progress Bar:** Segmented bars at the top indicating current media index.
*   **Header:** User info (Avatar, Name, View Count) and Close (X) button.
*   **Footer Actions:**
    *   **Reply:** Text input field to send a message.
    *   **Heart:** Like the story.
    *   **Send:** Share the story.
